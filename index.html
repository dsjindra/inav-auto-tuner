How to Decode .bbl / .bfl Files

  1. Download blackbox-tools from the INAV releases page.
  2. Open a terminal / command prompt in the folder containing your log file.
  3. Run the decode command (replace yourlog.bbl with your filename):
blackbox_decode yourlog.bbl

This generates yourlog.01.csv (one CSV per flight session stored in the log).

Windows: rename blackbox_decode_win64.exe → blackbox_decode.exe first.

Alternatively use INAV Blackbox Explorer → File → Export as CSV.
